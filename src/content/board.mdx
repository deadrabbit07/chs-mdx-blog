# 게시판 웹 애플리케이션

## 소개

사용자가 자유롭게 **글을 작성 / 조회 / 수정 / 삭제**할 수 있는 웹 기반 게시판 애플리케이션입니다.  
Express와 MySQL로 백엔드를 구성하고, Docker를 통해 손쉬운 실행 및 배포 환경을 구축했습니다.

## 기능

- 글 작성, 수정, 삭제 기능 제공
- 글 목록 및 상세 조회 기능
- MySQL을 활용한 게시글 데이터 저장
- Docker 기반 개발 및 배포 환경 구성

## 사용 기술

- Express.js
- MySQL
- Docker
- Node.js

## 개발 파트

- **BACKEND**: Node.js + Express.js 기반 RESTful API 서버 구현  
- **DATABASE**: MySQL 테이블 설계 및 쿼리 작성  
- **FRONTEND**: HTML/CSS 기반 정적 페이지 구성 및 JavaScript 로직 처리  
- **DEPLOYMENT**: Dockerfile 작성 및 이미지 빌드 후 실행
