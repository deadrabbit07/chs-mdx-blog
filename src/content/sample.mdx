# Portfolio


## 소개
개인 포트폴리오 웹사이트로, 본인의 프로젝트와 블로그를 손쉽게 관리하고, StackBlitz를 통한 코드 데모를 제공합니다.  
Next.js와 MDX를 활용하여 효율적인 콘텐츠 관리가 가능하도록 제작하였습니다.

## 기능
- 프로젝트, 블로그를 MDX 기반으로 관리
- 반응형 UI 제공
- StackBlitz를 통한 실시간 코드 실행
- Next.js를 활용한 페이지 라우팅
- Vercel을 통한 빠른 배포 환경 구축

## 사용 기술
- Next.js
- TypeScript
- Tailwind CSS
- MDX
- Vercel

## 개발 파트
- BACKEND: Next.js 기반 서버 사이드 렌더링
- FRONTEND: Next.js, Tailwind CSS
- [GitHub 바로가기](https://github.com/deadrabbit07/web-portfolio-serve)
