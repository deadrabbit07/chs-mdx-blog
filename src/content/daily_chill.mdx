# Daily Chill


## 소개

**Daily Chill**은 사용자의 감정에 맞춰 **조언과 칭찬**을 제공하고,  
**메모를 통해 하루를 정리**할 수 있도록 설계된 힐링 웹 애플리케이션입니다.  

TypeScript와 Tailwind CSS 기반으로 **빠른 개발과 일관된 UI**를 구현했으며,  
디지털새싹 APPJAM에서 **우수상**을 수상했습니다.

## 기능

- 사용자 기분에 맞는 chill한 조언 및 칭찬 제공
- chill guy와 chill한 배경에서 chill한 이야기
- 반응형 디자인 지원 (모바일/PC 모두 대응)
- Tailwind CSS 기반 깔끔한 인터페이스

## 사용 기술

- TypeScript
- React
- Tailwind CSS
- Vite
- Vercel

## 개발 파트

- **FRONTEND**: React + TypeScript로 전체 SPA 구성  
- **STYLE**: Tailwind CSS를 활용한 반응형 UI 디자인  
- **DEPLOYMENT**: Vercel을 통한 빠르고 안정적인 배포  

## 화면

- 메인 화면 (`감정 선택 후 Chill 배경`)
- 다이어리 작성 화면 (`일별 메모 입력 가능`)
- chill한 배경 화면(`chill guy와 이야기 가능`)
